# FightScene Backend API

Backend service for FightScene.gg built with Spring Boot, PostgreSQL, JPA, and Flyway.

## Tech Stack
- Java 21
- Spring Boot
- PostgreSQL
- Flyway
- Maven

## Running Locally
1. Create PostgreSQL database: `CREATE DATABASE fightscene;`
2. Set environment variable: `DB_PASSWORD=<yourpassword>`
3. Run: `mvn spring-boot:run`
